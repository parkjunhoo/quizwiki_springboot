## **📄 서비스 소개**

고등학생들을 위한 학습 플랫폼 입니다.


## 👨🏻‍💻 역할

프론트엔드&백엔드 [헤더,푸터,메인페이지, 문제 관련 기능, 오답노트 , 문제검색, 문제추천]

## 🛠 기술 스택

- Front-End
    - html js css(vanila)
    - AOS , Fabric(오답노트 부분 canvas) , sweetalert2 , 네이버 스마트에디터
- Back-End
    - Spring(boot)
    - Thymeleaf
- Database
    - MySQL
    - JPA
    - MyBatis
